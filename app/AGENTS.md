# AGENTS.md

## Project Overview

This is a small TypeScript project containing utility functions and unit tests.

## Tech Stack

* Language: TypeScript
* Runtime: Node.js 22+
* Test framework: Vitest

## Commands

* Install dependencies: `npm install`
* Run tests: `npm test`
* Lint: **Not configured**

## Coding Conventions

* Use TypeScript for all source files.
* Follow the existing coding style and formatting.
* Prefer simple, readable implementations over clever solutions.
* Keep functions focused on a single responsibility.
* When fixing a bug, add or update a regression test whenever appropriate.
* Avoid unnecessary dependencies or refactoring unrelated code.

## Guardrails

* Do not modify unrelated files.
* Preserve existing public function behavior unless explicitly required by the task.
* Ensure all tests pass before considering the task complete.
* Explain the implemented changes and the reasoning behind them.
* Keep changes as small and focused as possible.
