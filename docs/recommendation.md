# Рекомендація для команди (Task D)

## Рекомендований основний (primary) інструмент

**Claude Code (Claude CLI) з моделлю Haiku 4.5**

Чому: Task B показав, що Haiku 4.5 виконав баг-фікс найшвидше (2м 22сек) за мінімальну вартість ($0.0698), успішно знайшовши справжній баг, виправивши код та додавши регресійні тести за одну ітерацію без регресій. Task A підтверджує, що Claude Code забезпечує найвищу локальну агентну автономність з контекстним вікном до 1 млн токенів та ефективним кешуванням підказок. Task C показав, що Claude Code без проблем читає узагальнені AGENTS.md і CLAUDE.md.

## Рекомендований додатковий (secondary) інструмент

**GitHub Copilot CLI** — для сценаріїв, де важливі GitHub-інтеграція, гнучкість моделей або enterprise-функції (SSO, audit-логи, data residency).

Чому: Copilot CLI виконав задачу за 2м 30сек зі схожою вартістю, пропонує унікальну гнучкість вибору моделей (OpenAI + Anthropic) та безшовну інтеграцію з GitHub Issues/PR. Ідеальний для команд, повністю вбудованих у GitHub-екосистему, або коли потрібна можливість тестувати різні моделі-провайдери.

## Ключові компроміси (trade-offs)

- **Автономність vs. GitHub-інтеграція**: Claude Code вищу локальну автономність; Copilot — централізовану індексацію й enterprise-контроль. Вибір залежить від структури команди (локальна розробка vs. enterprise-процеси).

- **Вартість vs. якість на складних завданнях**: Haiku економний ($0.0698), достатній для більшості задач. Opus 4.8 коштує у 7–10 разів дорожче ($0.49), але потрібен тільки для справді складних архітектурних завдань.

- **Приватність vs. гнучкість**: Claude має строгу політику конфіденційності навіть на персональних планах; Copilot (Free/Pro) за замовчуванням передає дані для навчання. Для конфіденційних проектів — Claude; для відкритого коду — допускає Copilot.

- **Культура процесу**: Claude Code запитує дозвіл на комміти (рецензія); Copilot діє на автопілоті. Обирати за культурою команди.

## Ризики та застереження

- **Vendor lock-in**: дотримуватися крос-інструментного AGENTS.md, щоб обидва інструменти залишалися взаємозамінними. Стежити за змінами цін Anthropic.
- **Залежність від мережі**: Copilot потребує хмарної індексації; забезпечити локальні процедури для критичних операцій.
- **Compliance**: для чутливих даних використовувати корпоративні підписки обох провайдерів.
