## Висновок

Claude Code найсильніший для автономної багатоетапної роботи: ticket analysis, редагування, запуск тестів, subagents, hooks і MCP. Zed IDE найкращий, коли важливий editor-native workflow: швидкий review diff, LSP-контекст, робота в тих самих buffers і контроль над агентом прямо в IDE. OpenCode найгнучкіший і найвідкритіший варіант: open-source, багато моделей, локальні або enterprise gateway deployment, але якість сильніше залежить від конфігурації. Для мене, найбільш суттєвими відмінностями є зручність Zed IDE після довгої роботи з Intellij IDEA (можливість перенести знайомі shortcuts, зручний інтерфейс, нативна підтримка VIM motion без додаткових плагінів), а також вартість використання (підписки від Anthropic виглядають найбільш вигідними, якщо є бажання використовувати їх моделі)
