# Матриця можливостей AI-інструментів (Task A)

Інструменти, що порівнюємо:

1. **Claude Code** — CLI-агент від Anthropic
2. **OpenAI Codex CLI** — CLI-агент від OpenAI
3. **Cursor (Composer)** — AI-IDE (форк VS Code)

Дані станом на липень 2026.

## Оглядова таблиця

| Критерій | Claude Code | OpenAI Codex CLI | Cursor (Composer) |
|---|---|---|---|
| Вибір/якість моделі | Тільки Anthropic (Sonnet 5, Opus 4.8, Fable 5) | Тільки OpenAI (GPT-5.x-codex) | Мультивендор: Composer 2.5 + Anthropic/OpenAI/Google |
| Агентна автономність | Висока: субагенти, hooks, фонові задачі | Висока: режими схвалення, пісочниця | Висока: Composer-агент в IDE, паралельні агенти |
| Індексація кодової бази | Немає індексу, пошук на льоту | Немає індексу, пошук на льоту | Постійний семантичний індекс (embeddings) |
| Rules / commands / MCP | CLAUDE.md, skills, hooks, MCP | AGENTS.md, config.toml, MCP | .cursor/rules, AGENTS.md, MCP |
| Ціна та ліміти | Без безкоштовного плану, від $20/міс | Є безкоштовний доступ (ChatGPT Free) | Є безкоштовний план Hobby |
| Enterprise-функції | SSO, SOC 2, Bedrock/Vertex | SSO, SOC 2, Compliance Logs | SAML/SSO, SOC 2, примусовий Privacy Mode |
| Приватність / навчання на даних | Commercial/API — не навчається | Business/Enterprise — не навчається | Privacy Mode — не навчається |

## Детальний аналіз за критеріями

### 1. Вибір та якість моделі

- **Claude Code**: лише моделі Anthropic — Sonnet 5, Opus 4.8, Haiku 4.5, Fable 5 (топ-tier). Перемикання через команду `/model`.
- **Codex CLI**: лише моделі OpenAI — сімейство GPT-5.x-codex (флагман + mini-варіанти). Налаштовується рівень reasoning: low / medium / high.
- **Cursor**: найширший вибір — власна швидка модель Composer 2.5 (оптимізована під IDE) плюс frontier-моделі Anthropic, OpenAI та Google на вибір у кожному запиті.

### 2. Агентна автономність та multi-step execution

- **Claude Code**: повноцінний агент у терміналі — багатокрокове виконання, субагенти, фонові задачі, hooks, самостійний запуск тестів і команд.
- **Codex CLI**: агент у терміналі з трьома режимами схвалення (suggest / auto-edit / full-auto), пісочниця для безпечного виконання команд, cloud-задачі.
- **Cursor**: Composer-агент в IDE — багатокрокові правки по кількох файлах, паралельні агенти, вбудований термінал і браузер. Є також CLI-режим.

### 3. Індексація кодової бази

- **Claude Code**: постійного індексу немає — пошук на льоту (grep/glob) + контекст із `CLAUDE.md`. Працює на великих репо без попередньої підготовки, код нікуди не завантажується.
- **Codex CLI**: постійного індексу немає — досліджує репозиторій на льоту + контекст із `AGENTS.md`.
- **Cursor**: постійна семантична індексація (embeddings) усієї кодової бази — швидкий семантичний пошук, але код завантажується на сервери Cursor для побудови індексу.

### 4. Підтримка rules / commands / skills / MCP

- **Claude Code**: `CLAUDE.md` + `AGENTS.md`, slash-команди, skills, hooks, субагенти, повна підтримка MCP (як клієнт і як сервер). Найбагатша екосистема розширень.
- **Codex CLI**: `AGENTS.md`, конфіг `config.toml`, кастомні промпти, підтримка MCP.
- **Cursor**: правила у `.cursor/rules` (також читає `AGENTS.md`), кастомні команди, підтримка MCP.

### 5. Ціна та ліміти

- **Claude Code**: безкоштовного плану **немає**. Pro — $20/міс (достатньо для навчання), Max — $100–200/міс, або оплата API за токени. Ліміт — токен-бюджет із вікном 5 годин.
- **Codex CLI**: найдешевший старт — працює навіть із **безкоштовним** акаунтом ChatGPT (малі ліміти, вікно 5 годин). Plus $20/міс — комфортні ліміти, Pro $200/міс.
- **Cursor**: план Hobby — **безкоштовно назавжди** (обмежена кількість запитів) + ~2-тижневий тріал Pro при реєстрації. Pro $20/міс, Pro+ $60/міс.

### 6. Enterprise-функції (SSO / SOC 2 / data residency)

- **Claude Code**: плани Team ($25–30/seat) та Enterprise — SSO, SOC 2, аудит-логи, managed policy settings. Унікальне: деплой через AWS Bedrock / GCP Vertex, тобто дані залишаються у власній хмарі компанії.
- **Codex CLI**: ChatGPT Business/Enterprise — SSO, SOC 2, Compliance Logs Platform (централізований аудит), managed configs, політики для плагінів.
- **Cursor**: Teams $40–120/seat, Enterprise — SAML/SSO, SOC 2, примусовий Privacy Mode для всієї команди, адмін-контролі.

### 7. Приватність / політика навчання на даних

- **Claude Code**: дані API та Commercial-планів **не** використовуються для навчання; на споживчих планах навчання керується згодою користувача (можна вимкнути).
- **Codex CLI**: Business/Enterprise — **не** навчається на даних; на споживчих планах (Free/Plus) навчання ввімкнено за замовчуванням — треба вимкнути в налаштуваннях.
- **Cursor**: з увімкненим Privacy Mode — **жодного** навчання, zero-data-retention угоди з провайдерами моделей; без Privacy Mode Cursor може зберігати код і промпти та навчати на них власні моделі.

## Висновок

Найсуттєвіша різниця — у **ціні входу**: Codex CLI можна спробувати з безкоштовним акаунтом ChatGPT, Cursor має вічний безкоштовний план Hobby, а Claude Code потребує підписки від $20/міс. Друга ключова відмінність — **індексація проти приватності**: Cursor єдиний будує постійний семантичний індекс кодової бази (швидкий пошук по великому репо, але код іде на сервери), тоді як обидва CLI-агенти досліджують репозиторій на льоту. Третя — **гнучкість моделей**: Cursor дозволяє обирати моделі різних вендорів, а Claude Code і Codex прив'язані кожен до свого вендора. За агентною автономністю та екосистемою розширень (hooks, skills, субагенти, MCP) найбагатший — Claude Code. Для нашого кейсу (навчальний баг-фікс) усі три інструменти функціонально достатні.
