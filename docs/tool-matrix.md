## Висновок

Для цього зрівняння (Cursor vs OpenCode) найсуттєвіші відмінності:

- **Ціна та ліцензія:** Cursor Pro — $20/міс (або $16/міс при річній оплаті), закритий VS Code-форк; OpenCode Core — MIT, $0 + BYOK, або Go $10/міс.
- **Індексація:** Cursor має semantic search «з коробки»; OpenCode покладається на grep/glob/LSP без векторної індексації.
- **Інфраструктура агентів:** Cursor — Cloud Agents на віддалених VM (build/test); OpenCode — локальне виконання (CLI/desktop/IDE), без cloud VM.
- **Enterprise-готовність:** Cursor — SOC 2, SSO (Teams+), SCIM і data residency (Enterprise); OpenCode — Zen workspaces (beta) без формальних сертифікацій.
- **Приватність:** OpenCode не зберігає код локально і дозволяє BYOK; Cursor — Privacy Mode з політиками провайдерів (ZDR для більшості моделей).
- **Портативність контексту:** обидва підтримують `AGENTS.md`, що важливо для крос-tool baseline у Task C.
