# Матриця можливостей AI-інструментів (Task A)

> Скопіюйте у `docs/tool-matrix.md` і заповніть.

Інструменти, що порівнюєте: GitHub Copilot CLI, Claude Code (Claude CLI)

| Критерій | GitHub Copilot CLI | Claude Code (Claude CLI) |
|---|---|---|
| **Вибір/якість моделі** | Надає широкий вибір моделей від різних провайдерів (зокрема GPT-5 mini, Claude 4.5 Haiku, Claude Sonnet). Доступ до преміум-моделей типу Claude Opus відкривається на вищих тарифах (Pro+, Max). | Орієнтований виключно на флагманські моделі Anthropic (Claude Sonnet 4.6, Claude Opus 4.6/4.7/4.8). Має виняткову якість кодингу, глибокі логічні міркування та велике контекстне вікно (до 1 млн токенів). |
| **Агентна автономність та multi-step execution** | Підтримує режим "Agent mode" та кодувальних агентів. Здатний виконувати циклічні завдання, створювати та перевіряти пул-реквести, автоматично комітити локальні зміни або працювати безпосередньо в хмарі GitHub. | Побудований як повноцінне автономне агентне середовище в терміналі. Запускає автономні цикли пошуку помилок, редагування файлів, запуску тестів та виконання shell-команд. Має експериментальну підтримку multi-agent сесій ("teammates"). |
| **Індексація кодової бази** | Використовує централізовану хмарну індексацію на рівні репозиторіїв GitHub. Ефективно пов'язує контекст з Issues, PR, документацією проекту та корпоративними базами знань GitHub Spaces. | Спирається на локальну динамічну індексацію кодової бази "на льоту" (за допомогою інструментів на кшталт grep). Замість важких хмарних індексів використовує технологію Prompt Caching для швидкого завантаження великих контекстів. |
| **Підтримка rules / commands / skills / MCP** | Повна інтеграція з серверами Model Context Protocol (MCP) через GitHub MCP Registry. Підтримує файли інструкцій (.github/copilot-instructions), користувацькі навички (skills) та життєві цикли хуків (hooks). | Рідна (native) підтримка MCP, оскільки сам протокол створено Anthropic. Пропонує потужні вбудовані термінальні команди (`/search`, `/review`, `/cost`) та гнучке розширення можливостей через підключення зовнішніх MCP-інструментів. |
| **Ціна та ліміти** | Інтегрований у загальні плани GitHub Copilot: Free ($0, лімітований чат), Pro ($10/міс із пулом $15 в AI Credits), Pro+ ($39/міс, $70 credits), Max ($100/міс, $200 credits). Розрахунок ведеться за внутрішніми кредитними лімітами. | Доступний через індивідуальні підписки Claude Pro ($20/міс з лімітом оновлення кожні 5 годин), Max ($100-$200/міс) або Team Premium ($100/міс за користувача). Також підтримує пряму pay-as-you-go оплату через API за використані токени. |
| **Enterprise-функції (SSO/SOC2/data residency)** | Глибока інтеграція на рівні організації з планами Copilot Business та Enterprise. Нативна підтримка SSO, SAML, SOC2 Type II, детальних аудит-логів корпоративної активності та гнучких політик розміщення даних. | Забезпечується через плани Claude Team Premium та Enterprise. Включає централізований білінг, панелі адміністратора, підтримку SSO/SCIM, а також відповідність стандартам SOC2/HIPAA через корпоративні платформи Anthropic, AWS Bedrock або Azure. |
| **Приватність / політика навчання на даних** | На тарифах Business та Enterprise діє повна заборона на використання даних організації для навчання. На персональних планах (Free/Pro) користувачі повинні вручну вимкнути опцію передачі даних у налаштуваннях. | Сувора політика конфіденційності Anthropic: дані комерційних підписок (Team/Enterprise) та запити через API ніколи не використовуються для навчання моделей. В індивідуальних планах Pro приватність також увімкнена за замовчуванням. |

## Висновок

Claude Code демонструє значну перевагу в глибокій контекстній автономності, швидкості локальної роботи та інтеграції з терміналом завдяки великому вікну контексту Anthropic та ефективному кешуванню підказок. Водночас GitHub Copilot CLI виграє за рахунок гнучкості вибору моделей різних провайдерів та безшовної хмарної інтеграції з екосистемою GitHub (PR, Issues, корпоративні бази знань). Найбільш суттєвою відмінністю є підхід до контексту: Copilot CLI ідеально підходить для розробки у великих хмарних enterprise-репозиторіях, тоді як Claude Code є потужнішим інструментом для інтенсивних локальних сесій рефакторингу та автономного виправлення багів безпосередньо в консолі.

Якщо повністю виключити хмарну інтеграцію з GitHub, протистояння в консолі стає битвою двох абсолютно різних філософій. Claude Code беззаперечно перемагає в номінації "Автономний локальний інженер" — завдяки гігантському контекстному вікну, Prompt Caching та вбудованому агентному циклу він здатен самостійно вирішувати комплексні завдання (наприклад, повний локальний рефакторинг чи міграцію бібліотек), але це коштує значно дорожче. З іншого боку, GitHub Copilot CLI у суто локальному режимі виступає як ідеальний, швидкий та економічний "асистент-штурман" за $10/міс. Він надає унікальну свободу вибору між моделями OpenAI та Anthropic і чудово підходить для точкових локальних правок, генерації shell-команд та інтерактивного дебагу без значних фінансових витрат.
