# Матриця можливостей AI-інструментів (Task A)

Інструменти, що порівнюються: **Claude Code у складі Claude Pro** та **ChatGPT Plus**.

Область порівняння: індивідуальні плани для користувача в Україні; ціни наведені за публічною ціною для США без локальних податків.
Дата актуальності: **13 липня 2026 року**.

## Висновок

Для мого сценарію найбільша практична відмінність — доступ до локального workspace та рівень автономності. Claude Code виконав повний цикл Task B: знайшов файли, змінив реалізацію і тести, обійшов проблему PowerShell через `npm.cmd` та підтвердив 8/8 успішних тестів. ChatGPT Plus краще підходить для аналізу, пояснень і підготовки документації, але в протестованій конфігурації не забезпечував такий самий локальний агентний цикл. Для індивідуальних планів обидва інструменти коштують 20 USD на місяць за публічною ціною для США, а точні usage limits постачальники не гарантують одним фіксованим числом.
