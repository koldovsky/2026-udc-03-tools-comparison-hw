# Нотатки до AGENTS.md (Task C)

## Що додано/змінено в `app/AGENTS.md`

- Прибрано прив'язку до одного інструменту: заголовок `# Cursor Rules` →
  `# AGENTS.md` з явною приміткою, що baseline крос-tool (Copilot, Gemini,
  Cursor, Claude Code тощо).
- Додано секцію **Stack**: TypeScript (ESM, `"type": "module"`), Node 22+, Vitest,
  без окремого build-кроку.
- Додано секцію **Commands** з реальними скриптами: `npm test`, `npm run test:watch`,
  `npm run typecheck`; чесно зафіксовано `lint: не налаштовано` (без вигаданої команди).
- Винесено конкретні **конвенції коду**: стиль існуючого файлу, pure/functional,
  2 пробіли + подвійні лапки + крапки з комою, ESM-імпорти з розширенням `.js`,
  тест на кожну зміну поведінки.
- Посилено **guardrails**: не міняти публічні сигнатури `slugify`/`truncate`/`parseTags`,
  тримати `npm test` і `typecheck` зеленими, завершальний newline, без секретів/PII.

| Gemini (ця IDE) | Так | Повністю відтворив усі секції `AGENTS.md`: Vitest, `npm test`/`npm run typecheck`, `lint: не налаштовано`, `.js`-імпорти, guardrails (не міняти сигнатури, newline в кінці файлу). Нічого не вигадав |

## Промпт верифікації

> Опиши конвенції та команди цього проєкту (стек, build/test/lint, стиль коду), спираючись на AGENTS.md. Нових файлів не створюй.

## Результат

- **Copilot:** дотримався `AGENTS.md` без додаткових нагадувань — назвав Vitest,
  команди `npm test`/`npm run typecheck`, відсутність лінтера, `.js`-імпорти та
  заборону міняти публічні сигнатури.
- **Gemini:** так само підхопив `AGENTS.md` без налаштувань — точно перелічив стек,
  усі команди, `lint: не налаштовано`, конвенції коду (2 пробіли, подвійні лапки,
  `.js`-імпорти) і guardrails. Формулювання майже дослівно збіглися з файлом.

## Висновок

Одного `AGENTS.md` вистачило як крос-tool baseline: **обидва** інструменти
(Copilot і Gemini) підхопили його без додаткових налаштувань чи tool-specific
файлів і однаково точно відтворили стек, команди та guardrails — включно з
чесним `lint: не налаштовано`. Портативність контексту підтверджена.
