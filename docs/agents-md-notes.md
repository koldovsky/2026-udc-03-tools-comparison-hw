## Результат

Claude Code відповів, що lint-команда не налаштована, і перелічив доступні команди (`npm test`, `npm run test:watch`, `npm run typecheck`). Це відповідає як `package.json`, так і вказівці в `AGENTS.md`, що lint у проєкті відсутній.

Для ChatGPT було змодельовано той самий сценарій із наданим вмістом `AGENTS.md`. Після отримання цього контексту інструмент також визначає, що lint не налаштований, однак, на відміну від Claude Code, не може автоматично прочитати локальний файл або перевірити структуру проєкту без явного надання контексту.

## Висновок

Один файл `AGENTS.md` добре працює як універсальний baseline для різних AI-інструментів. Проте агентні інструменти на кшталт Claude Code можуть автоматично використовувати його разом із контекстом відкритого workspace, тоді як ChatGPT потребує, щоб цей файл або відповідні інструкції були явно передані в діалозі.
