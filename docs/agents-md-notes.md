# Нотатки до AGENTS.md (Task C)

## Що додано/змінено в `app/AGENTS.md`

- **Прибрав прив'язку до одного інструменту:** заголовок був `# Cursor Rules`
  зі згадкою Cursor — узагальнив до `# AGENTS.md` як крос-tool baseline
  (Claude Code, Junie, Cursor, Copilot, …).
- **Додав секцію «Стек»:** TypeScript strict (+`noUncheckedIndexedAccess`),
  ESM (`"type": "module"`, ES2022/ESNext/Bundler), тести на Vitest, без
  runtime-залежностей.
- **Додав секцію «Команди»:** `npm test` (`vitest run`), `npm run test:watch`,
  `npm run typecheck` (`tsc --noEmit`); **build — не налаштовано**,
  **lint — не налаштовано** (не вигадував неіснуючих команд).
- **Формалізував 5 конвенцій коду** (раніше були розмиті «clean/readable»):
  чисті функції з named exports, функціональний стиль, тести поруч (`*.test.ts`),
  ESM-імпорти з розширенням `.js`, коментарі лише для неочевидного.
- **Розширив guardrails:** тести зелені, стабільні публічні сигнатури,
  фікс = фікс + регресійний тест, без нових залежностей без причини.

його без tool-specific файлів (`.cursor/rules`, окремий `CLAUDE.md` тощо) і
дотрималися однакових конвенцій. Ключове — писати факти чесно (явно «lint не
налаштовано»), бо інструменти дослівно повторюють baseline і за наявності
вигаданої команди відтворювали б помилку.
