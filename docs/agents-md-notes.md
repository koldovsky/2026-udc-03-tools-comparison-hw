# Нотатки до AGENTS.md (Завдання C)

Цей документ фіксує, що змінено в app/AGENTS.md, як перевірити зміни мінімально
у двох інструментах, і надає приклади промптів та кроки верифікації для
грейдерів.

## Що додано/змінено у app/AGENTS.md

- Узагальнено оригінальні правила, прив'язані до Cursor, у крос‑інструментну
  базову інструкцію "Agents Guidelines" — формулювання та приклади, що
  підходять як для CLI‑асистентів, так і для web‑інтерфейсів.
- Додано лаконічний «шаблон першого промпта» (опис задачі, обмеження, команди
  для верифікації, очікуваний результат, формат виводу).
- Включено команди верифікації (cd app && npm test) і вказівку демонструвати
  цикл "failing → passing" при додаванні регресійного тесту.
- Додано нейтральні до інструменту нотатки щодо комітів і принципу "змінюй
  мало й сфокусовано".
