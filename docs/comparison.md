## Висновок

На цій задачі Claude Code продемонстрував повний робочий цикл: з першого промпту знайшов причину бага, змінив код, додав регресійні тести та підтвердив результат запуском `npm.cmd test` — 8/8 тестів пройшли. ChatGPT також запропонував логічно правильне виправлення, але його результат оцінювався лише за відповіддю: зміни не були окремо застосовані та перевірені локальним запуском тестів. Найсуттєвішою відмінністю стала здатність Claude Code самостійно працювати з workspace і виконувати перевірки.
